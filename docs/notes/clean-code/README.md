# Clean Code Notes

Learning notes adapted from Robert C. Martin's *Clean Code* for the agentic era.

These notes are not repository policy and are not agent instructions.
They are a staging area for:

- what the book argues
- what we want to keep
- what we want to reject or soften
- what we want to adapt for human and AI-assisted coding
