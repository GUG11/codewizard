# Technical Article Writing Skill Tests

This folder contains scenario tests for `plugins/codewizard/skills/technical-article-writing`.

Run with Codex:

```bash
codex "Run agents/tests/skills/technical-article-writing/runner.md"
```

The runner compares `with-skill` and `without-skill` variants to evaluate whether the skill improves source-backed links, causal storytelling, and non-defensive revision behavior under adversarial user feedback.
